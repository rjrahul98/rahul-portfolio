# Rahul's Portfolio

A modern, interactive portfolio website built with Next.js 16, React 19, and Three.js.

## 🚀 Features

- **3D Graphics**: Interactive 3D hero sections powered by Three.js and React Three Fiber
- **Modern Stack**: Built with Next.js 16, React 19, and TypeScript
- **Responsive Design**: Fully responsive with Tailwind CSS 4
- **Smooth Animations**: Framer Motion for fluid transitions
- **Project Showcase**: Interactive project cards with detailed information
- **Contact Form**: Easy way to get in touch

## 🛠️ Tech Stack

- **Framework**: Next.js 16.1.1
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Animation**: Framer Motion
- **Package Manager**: pnpm

## 🌐 Live Site

Visit the live portfolio at: [https://rjrahul98.github.io/rahul-portfolio/](https://rjrahul98.github.io/rahul-portfolio/)

## 💻 Local Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 📦 Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the master branch.
